Draw beliefs in OpenGL scene according to boolean flags.

Parameters:
centerPointPositionOffset for all OpenGL locations. This real-world location corresponds to OpenGL location 0,0,0.
estLatToMConvEstimated conversion from latitude to meters for the area. Used to map latitudes in meter space
estLonToMConvEstimated conversion from longitude to meters for the area. Used to map longitudes in meter space
drawAgentPositionBeliefIf true, display UAV agent in OpenGL
drawGCSAgentPositionBeliefIf true, display GCS agent in OpenGL
drawCircularOrbitBeliefIf true, display circular orbit in OpenGL
drawBiopodDetectionBeliefIf true, display biopod detections in OpenGL (from CloudDetectionBelief, not ParticleDetectionBelief)
anacondaDetectionBeliefIf true, display Anaconda detections in OpenGL (from CloudDetectionBelief, not AnacondaDetectionBelief)
uavDisplayFromPicTelIf true (and if drawAgentPositionBelief is true), UAV display should be done using PiccoloTelemetryBelief data instead of AgentPositionBelief data
