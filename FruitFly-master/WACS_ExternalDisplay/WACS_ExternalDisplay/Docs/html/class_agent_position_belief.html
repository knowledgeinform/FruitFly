bool AgentPositionBelief::addAgentPosition ( AgentPosition *  newPosition)

Add newPosition to AgentPosition history list and store it as the current position.

Adding newPosition is dependent on checking timestamp of newPosition against the last received position. If message is not new, delete the message. If the message is new, add newPosition to history.

Parameters:
newPositionAgentPosition to add to the history
Returns:
True if the newPosition was new data and was added to the list. False if newPosition was not new and not added.
