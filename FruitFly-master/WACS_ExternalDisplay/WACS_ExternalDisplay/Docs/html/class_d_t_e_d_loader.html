Loads a DTED file from file and adds it to dtedObject. If first file being loaded, dtedObject is populated with general data of the region to be loaded.

Parameters:
filenameFilename of DTED file to merge into dtedObject
minLatBottom-left corner of region of interest of dtedObject. Latitude decimal degrees
minLonBottom-left corner of region of interest of dtedObject. Longitude decimal degrees
maxLatUpper-right corner of region of interest of dtedObject. Latitude decimal degrees
maxLonUpper-right corner of region of interest of dtedObject. Longitude decimal degrees
spacingInterval to load data. If 2, load every other point. If 3, load every 3rd point. Used to downsample large areas for memory preservation
dtedObjectDTEDObject to fill with DTED elevation data. Will have sizes and limits set the first time, then elevations udpated with each subsequent call with different files
firstTileIf true, this if the first tile generating data for dtedObject. dtedObject needs its limits and sizes established
Returns:
DTEDObject pointer to DTED object
